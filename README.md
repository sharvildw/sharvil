# Premium Developer Portfolio

An ultra-modern, production-grade full-stack portfolio platform built for a Computer Engineering student. It uses a modern decoupled architecture designed to impress recruiters, startups, and clients.

## Tech Stack

- **Frontend:** React.js, Vite, TypeScript, Tailwind CSS, Framer Motion, shadcn/ui
- **Backend:** Node.js, Express.js, TypeScript, MongoDB, JWT Authentication
- **Design:** Glassmorphism, modern typography, cinematic transitions, animated components

## Getting Started

### 1. Clone the repository

### 2. Setup the Backend
\`\`\`bash
cd backend
npm install
npm run dev
\`\`\`
*(Ensure you have a MongoDB instance running locally or provide a `MONGO_URI` in your `.env` file.)*

### 3. Setup the Frontend
\`\`\`bash
cd frontend
cd frontend
npm run dev
\`\`\`

## Features

- **Hero Section:** Animated typing, glowing backgrounds, and particle effects.
- **Projects Showcase:** Interactive project cards with hover tilt effects and technical stack tags.
- **Admin Dashboard (API Structure Built):** Secure JWT authentication, RESTful APIs for managing projects and skills.
- **Responsive & Accessible:** Fully responsive across all devices with clean, semantic HTML.
